# 4x4產生器
4 乘 4 共 16 張圖的圖片產生器。

## 功能

### 圖片
* 上傳：強制拉開比例為正方形。
* 輸出：`1024px`*`1024px` 圖檔

### 文字樣式
* 文字內容
* 是否顯示
* 顏色
* 字體大小
* 字重
* google 字型

## 開發工具
* HTML
* SCSS
* Javascript
* jQuery
* Vue.js
